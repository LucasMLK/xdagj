## 重构概述

### 目标
将 XdagJ 从 v1.0 架构（Block + Address）迁移到 v5.1 架构（Block + Transaction + Link）。

### 核心变更
- **删除 Block 类** - 旧的块结构（包含 Address 对象）
- **引入 Block 类** - 新的不可变块结构（使用 Link 引用）
- **引入 Transaction 类** - 独立的交易对象
- **引入 Link 类** - 轻量级引用（替代 Address）
- **删除 XdagStats** - 用 ChainStats（不可变）替代
- **删除 XdagExtStats** - hashrate跟踪移至 ChainStats

## 架构变更总结

### v1.0 → v5.1 对比

| 组件 | v1.0 (旧) | v5.1 (新) | 变更原因 |
|------|-----------|-----------|----------|
| **Block** | Block (可变) | Block (不可变) | 简化、线程安全 |
| **Transaction** | 嵌入Block | Transaction (独立) | 解耦、可验证 |
| **References** | Address (含金额) | Link (仅哈希+类型) | 轻量化 |
| **Statistics** | XdagStats (可变) | ChainStats (不可变) | 线程安全 |
| **Import** | tryToConnect(Block) | tryToConnect(Block) | 简化验证 |
