## 🎯 Quick Start Path

### For Developers
1. Start with [DevNet Tutorial](./devnet-tutorial-en.md)
2. Learn wallet operations: [CLI Wallet Guide](./cli-wallet.md)
3. Progress to [TestNet Tutorial](./testnet-tutorial-en.md)

### For Miners
1. Read [CLI Wallet Guide](./cli-wallet.md)
2. Follow [TestNet Tutorial](./testnet-tutorial-en.md)
3. Configure mining setup

### For Users
1. Start with [CLI Wallet Guide](./cli-wallet.md)
2. Join TestNet: [TestNet Tutorial](./testnet-tutorial-en.md)
