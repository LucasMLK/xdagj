**创建文件**: `src/test/java/io/xdag/e2e/TransferE2ETest.java`

**测试用例**:
```java
@Test
public void testCompleteTransferFlow() {
    // 场景：Alice向Bob转账100 XDAG

    // 1. 初始化
    //    - Alice余额: 1000 XDAG
    //    - Bob余额: 0 XDAG

    // 2. Alice创建Transaction
    Transaction tx = Transaction.builder()
        .from(aliceAddress)
        .to(bobAddress)
        .amount(XAmount.of(100, XUnit.XDAG))
        .nonce(1)
        .fee(XAmount.of(100, XUnit.MILLI_XDAG))
        .build();

    // 3. Alice签名
    Transaction signedTx = tx.sign(aliceKey);
    assertTrue(signedTx.verifySignature());

    // 4. 保存Transaction
    transactionStore.saveTransaction(signedTx);

    // 5. 创建BlockV5
    BlockV5 block = BlockV5.builder()
        .header(...)
        .links(Lists.newArrayList(Link.toTransaction(signedTx.getHash())))
        .build();

    // 6. 连接到区块链
    ImportResult result = blockchain.tryToConnect(block);
    assertEquals(ImportResult.IMPORTED_BEST, result);

    // 7. 应用块
    blockchain.applyBlock(true, block);

    // 8. 验证余额变化
    //    - Alice余额: 899.9 XDAG (1000 - 100 - 0.1)
    //    - Bob余额: 100 XDAG

    // 9. 验证nonce递增
    assertEquals(2, addressStore.getTxQuantity(aliceAddress).toLong());
}
```

**预计时间**: 3-4小时
**覆盖**: 完整的v5.1交易生命周期
